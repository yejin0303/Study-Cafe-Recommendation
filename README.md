# ☕ Study Cafe Recommendation

A machine learning-based study cafe recommendation system that analyzes cafe reviews and environmental factors to recommend cafes suitable for studying.

---

## 📖 Overview

This project aims to identify study-friendly cafes by analyzing cafe information and user reviews. The recommendation score is calculated by combining study-related review features, ratings, business hours, and distance using a weighted scoring approach. Multiple machine learning models were trained and evaluated to identify the most effective recommendation model.

---

## ✨ Key Features

- 📍 Collect and preprocess cafe information and review data
- 📝 Analyze study-related review features
- 🤖 Train and evaluate machine learning models
- 📊 Calculate recommendation scores using weighted factors
- 📈 Compare model performance
- ☕ Recommend cafes optimized for studying
